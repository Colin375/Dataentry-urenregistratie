# Dataentry-urenregistratie gebruiken
Om deze file te gebruiken moet je eerst php geïnstalleerd hebben dat je ook in command prompt kan gebruiken, dan in het command prompt van php kan je de file starten door "php urenregistratie.php" te typen als je in dezelfde map zit, als dit niet werkt maar php wel, dan verander je urenregistratie.php naar het hele pad inclusief het bestaand naam dat je probeert te gebruiken.

# Wat is dit?
Dit project is om gemakkelijk csv bestanden via een php terminal te maken om een datum, project naam en aantal gewerkte uren aan het project op te slaan om makkelijk te importeren naar programma's dat csv bestanden ondersteunen (bijvoorbeeld excel).
