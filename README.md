# Running Dataentry-urenregistratie 
To run this file you first need a valid php installation which can be run in cmd, then in the php command prompt you just run the file by typing "php urenregistratie.php" if in the same folder, otherwise change urenregistratie.php to be the entire path including file name to the file you are trying to run.

# What is this?
This project is to easily make csv files via a php terminal to save a date, project name and amount of hours worked on the project for easy import into programs that support csv (for example excel).
